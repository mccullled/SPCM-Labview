This software can be used to operate Thorlabs PM100D, PM100USB and PM100A optical power/energy meters

The software folder contains the following programs:

1) PM100D Utility Installer
   Remote Control of the PM100D/A/USB, data logging and visualizing.
   VISA420runtime must be installed on the target systm.

2) PM100D Utility Executable
   Remote Control of the PM100D/A/USB, data logging and visualizing.
   On the target system the LabVIEW 8.5.1 runtime engine and VISA420 runtime must be istalled.
   A LabVIEW installation is not necessary

3) PM100D Utility Source Distribution
   Remote Control of the PM100D/A/USB, data logging and visualizing.
   LabVIEW 8.5.1 source code - LabVIEW 8.5.1 or higher required.

4) PM100D Simple Example
   Example to control the PM100D/A/USB in LabVIEW with SCPI commands.´
   LabVIEW 8.5.1 source code - LabVIEW 8.5.1 or higher required.
